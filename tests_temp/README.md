# Tests Temp

Pasta para testes temporários e experimentos.

## Propósito

Esta pasta é destinada a scripts de teste e debugging temporários que não fazem parte da suite de testes oficial do projeto.

## Uso

- Scripts de teste rápido
- Debugging de funcionalidades específicas
- Experimentos temporários
- Validações pontuais

## ⚠️ Importante

Os arquivos nesta pasta **não são versionados** e podem ser removidos a qualquer momento. Não coloque aqui código importante ou permanente.

Para testes oficiais, use a pasta `tests/`.
